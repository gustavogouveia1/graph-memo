# Template de Nota Atomica

```md
# Titulo da Nota

- tipo: #type/decision
- status: #status/in-progress
- dominio: #domain/backend
- risco: #risk/medium
- atualizado_em: YYYY-MM-DD

## Contexto

Situacao objetiva em 3-5 linhas.

## Declaracao

Uma afirmacao tecnica clara e verificavel.

## Evidencias

- Link para codigo/PR/issue/ADR
- Dado ou metrica relevante

## Relacoes (wikilinks)

- [[Entity:Nome]]
- [[Flow:Nome]]
- [[ADR-XXX]]

## Proximos passos

- Acao 1 com dono e prazo
```
